# CraftGame
a minecraft-like game programmed in Java with LWJGL 2.
The game is in beta now. So if you have some suggestion, just leave in the comment.
****

#### How to build and run

* Windows

To build, just type
> gradlew.bat build

then
> gradlew.bat run

There you go!
